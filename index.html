Summary Latitude vs X

In this project we analyze different weather attributes based on latitude position. We pulled data from the OpenWeatherMap via their API and compiled data of over 500 randomly selected cities.

Once the data was received, we used the Matplotlib library to visually show the weather attributes vs the latitude. We analyzed four factors which included: temperature, cloudiness, wind speed and humidity.
